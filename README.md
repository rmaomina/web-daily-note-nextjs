# 시시콜콜, Daily note with NextJS

## 🚀 기능

어디에도 적지 못하는 시시콜콜한 일상의 기록을 써내려갑니다.
노션 API를 활용한 사이드 프로젝트입니다.

1. 노션 api로 database를 조회합니다.
2. 노션 페이지 Id 별로 상세 페이지로 이동할 수 있습니다.
